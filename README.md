## Tech Stack

**Client:** Pure HTML

**Server:** Node, Express


## Environment Variables

To run this project, you will need to install the following environment

`socket.io`

`express`


## Run Locally

Clone the project

```bash
  git clone https://github.com/Sirius-Star42/nodejs_chat_app.git

  Go to the project directory

  cd main folder
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run server
```
